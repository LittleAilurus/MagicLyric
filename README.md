# MagicLyric
## 前情提要
这是一个平平无奇的歌词展示工具，主要用于直播。
源于本人在直播的时候，感觉没有很好用的歌词实时动效工具，想着要不就自己手搓一个吧。
鄙人不才，只会写点垃圾代码，请轻喷。本着先走后跑步，前期能用就行的原则，欢迎大家一起协作指导。

## 技术细节
本项目使用Unreal Engine 5.7+ 版本进行开发，主要逻辑均使用蓝图进行开发。
本项目素材部分来自可商用的素材库，部分来自自研素材。

## 研发路线
计划中的功能：
1.支持Now Playing 2 的websocket歌词输入，实现歌词同步展示
2.实现若干歌词的动态效果编排
3.支持spout2输出整体画面
4.支持通过spout2输入VTS的画面
5.支持通过面板或快捷键进行实时的歌词动态效果控制
6.若干歌词动态效果控制的预设
7.整体的Post-FX特效
8.各种运镜、镜头特效，并支持通过面板或快捷键控制
9.自定义2D/3D场景（背景）
10.支持多场景，并能通过面板或快捷键进行快速切换
